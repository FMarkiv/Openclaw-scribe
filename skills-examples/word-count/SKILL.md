# word-count

Use this skill to count words, lines, and characters in files.

## When to use
- When the user asks about file sizes or word counts
- When analyzing document or code statistics

## How to use
- Provide the `path` to a file or directory
- Set `recursive` to "true" to count all files in a directory tree

## Output format
The output lists each file with its line count, word count, and
character count, followed by a total if multiple files are counted.
